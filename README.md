# Django REST API
### git clone directory
``` git clone https://github.com/django-restframewrk```
### change Directory
``` cd django-restframewrk ```
### activate virtualenv
``` source bin/activate ```
### install Django and Django Restframework
```pip install django```

```pip install djangorestframework```
### Run server
```python manage.py runserver ```
